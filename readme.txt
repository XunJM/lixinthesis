请先打开main.pdf, 并仔细阅读。如果你发现该模板有更新，一般情况下你只需下载lixinthesis.cls并覆盖原有的lixinthesis.cls。
使用ctex打开时，要用utf-8格式打开。而且ctex似乎不能编译。请使用本文推荐的环境。
